Phiên bản 1.0.0 (Premium Planner)

Một ứng dụng lập lịch trình cá nhân được tối ưu hóa tỉ mỉ, giúp quản lý thời gian và sắp xếp công việc hiệu quả trong ngày. Tích hợp sâu với Telegram API giúp đảm bảo bạn không bao giờ bỏ lỡ các công việc quan trọng bằng cách đẩy thông báo tự động.

Các tính năng nổi bật:

  • Hệ thống nhắc nhở tự động theo thời gian thực (định kỳ mỗi giây).
  • Tích hợp Telegram Bot API gửi nhắc nhở và tự động xóa tin nhắn nhắc cũ khi hoàn thành công việc để giữ cuộc trò chuyện sạch sẽ.
  • Chế độ hiển thị Lịch Tháng trực quan với các chấm chỉ báo màu của danh mục.
  • Thiết kế vẽ tay hoàn toàn bằng mã CSS thuần túy, không tải bất kỳ bộ icon hay thư viện ngoài.
  • Hỗ trợ xuất/nhập file cấu hình định dạng JSON linh hoạt.
  • Chế độ tối thông minh tự động đổi màu dịu mắt.
